View detailed information for each API call with filtering by time, model, token, and more. Click "Logs" in the left sidebar, or visit `/console/log` directly. Regular users can only see their own call records.

![Personal log page](/assets/guide/feature-guide/log-list.png)

Each row in the log list represents one API call, showing the call time, model used, token count consumed, quota deducted, and call status.

## Search & Filter

1. Click the "Filter" button at the top of the log page to expand the filter options

![Filter options expanded](/assets/guide/feature-guide/log-filter-open.png)

2. Set any of the following conditions:
   - **Time range**: Select start and end dates
   - **Model**: Enter a model name keyword
   - **Token name**: Select or enter a token name
3. Click "Query" — the list refreshes to show filtered results

![Filtered log list](/assets/guide/feature-guide/log-filtered.png)

## Data Statistics

1. Click "Dashboard" in the left sidebar, or visit `/console` directly

![Personal data statistics chart](/assets/guide/feature-guide/dashboard-chart.png)

2. The dashboard page shows daily API call volume and quota consumption trends as a line or bar chart
3. Hover over the chart to view detailed data for a specific date
