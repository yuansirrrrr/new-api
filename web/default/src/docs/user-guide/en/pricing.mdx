The pricing page shows the billing multipliers for all available models. No login required.

1. Click "Pricing" in the left sidebar, or visit `/pricing` directly

![Pricing page overview](/assets/guide/feature-guide/pricing.png)

2. The page lists all available models — each row shows the model name, input price, and output price
3. Use the search box at the top of the page to enter a model name keyword and quickly locate a specific model's price

## Price Explanation

- **Input price**: Quota consumed per 1K input tokens
- **Output price**: Quota consumed per 1K output tokens
- Actual consumption = token count ÷ 1000 × unit price
- Users in different groups may have different billing multipliers — check your balance change on the top-up page for the actual deduction
