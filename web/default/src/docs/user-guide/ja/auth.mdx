title: 登録とログイン

---

パスワード登録および複数のサードパーティ OAuth ワンクリックログインに対応しています。初めてご利用の方はまず登録を完了してください。

## ログイン

### パスワードログイン

1. プラットフォームのホームページを開き、右上の「ログイン」ボタンをクリックするか、`/login` に直接アクセスしてください

![ログインページ](/assets/guide/feature-guide/login.png)

2. ユーザー名とパスワードを入力し、「ログイン」をクリックします

### サードパーティ OAuth ログイン

サードパーティアカウントでログインする場合は、ページ下部の対応するプラットフォームのアイコン（GitHub、Discord、LinuxDO など）をクリックし、サードパーティの認証ページで認証を完了すると自動的にログインされます

### パスワードを忘れた場合

パスワードを忘れた場合は、ログインページの「パスワードを忘れた」リンクをクリックし、登録済みのメールアドレスを入力してください。システムがリセットリンクを送信します。リンクをクリックして新しいパスワードを設定すると、元のパスワードは即座に無効になります。

## 登録

### 登録手順

1. ログインページの「登録」リンクをクリックするか、`/register` に直接アクセスしてください

![登録ページ](/assets/guide/feature-guide/register.png)

2. ユーザー名とパスワードを入力します
3. メールアドレスを入力し「認証コードを送信」をクリックして、受け取った認証コードを入力します
4. 「登録」をクリックしてアカウント作成を完了します。登録成功後、自動的にホームページにリダイレクトされます
