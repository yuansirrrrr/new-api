---

サブスクリプションは定期購入のパッケージです。購入後、有効期間中はパッケージ内のクォータや特典を利用できます。左側ナビゲーションの「サブスクリプション」をクリックするか、`/console/subscription` に直接アクセスしてください。

![サブスクリプションプラン一覧](/assets/guide/feature-guide/subscription-plans.png)

サブスクリプション一覧には、購入可能なすべてのパッケージが表示されます。パッケージ名、価格、有効期間、含まれるクォータなどの情報が含まれます。

サブスクリプションパッケージは日次、週次、月次などのサイクルで利用できる定期クォータバンドルを提供します。

## サブスクリプションの購入

1. サブスクリプションページで利用可能なプランを確認し、各プランの価格、有効期間、含まれるクォータを確認します
2. 希望するプランの「購入」ボタンをクリックします

3. 支払い方法を選択し「支払いを確認」をクリックします
4. 支払い完了後、自動的に戻り、サブスクリプションステータスが有効に更新されます

## 現在のサブスクリプション状況の確認

購入後、サブスクリプションページの上部に現在のプランの詳細が表示されます：

- プラン名と有効期限
- プラン内の残りクォータ
- 自動更新トグル：プランの有効期限が切れた後に自動更新するかどうかを設定できます
