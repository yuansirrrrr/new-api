---
title: タスク管理
description: Midjourney 画像生成や Suno 音楽生成などの非同期タスクを管理
---

Midjourney 画像生成や Suno 音楽生成などの非同期タスクのステータスと結果を管理します。左側ナビゲーションの「タスク」をクリックするか、`/console/task` に直接アクセスしてください。

![タスク一覧ページ](/assets/guide/feature-guide/task-list.png)

タスク一覧には、送信されたすべての非同期タスクが表示されます。タスク ID、タイプ、ステータス、送信時刻、完了時刻などの情報が含まれます。

タスクページには Midjourney や Suno を含むすべての非同期生成タスクのステータスが表示されます。

タスクステータスの説明：

| ステータス | 説明 |
| --- | --- |
| `PENDING` | 処理待ち |
| `IN_PROGRESS` | 生成中 |
| `SUCCESS` | 生成完了 |
| `FAILURE` | 生成失敗（クォータは返還済み） |

## Midjourney タスクの詳細

タスク一覧で SUCCESS ステータスの Midjourney タスクをクリックして詳細を確認してください。

Midjourney タスク完了後、アップスケール（U）やバリエーション（V）などの二次操作を直接実行できます。
