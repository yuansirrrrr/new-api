各 API 呼び出しの詳細情報を確認できます。時間、モデル、トークンなどでフィルタリングに対応しています。左側ナビゲーションの「ログ」をクリックするか、`/console/log` に直接アクセスしてください。一般ユーザーは自分の呼び出し記録のみ確認できます。

![個人ログページ](/assets/guide/feature-guide/log-list.png)

ログ一覧の各行は 1 回の API 呼び出しを表し、呼び出し時刻、使用モデル、消費トークン数、クォータ控除額、呼び出しステータスが表示されます。

## 検索とフィルタリング

1. ログページ上部の「フィルター」ボタンをクリックしてフィルター条件を展開します

![フィルター条件展開状態](/assets/guide/feature-guide/log-filter-open.png)

2. 以下の条件を設定できます：
   - **時間範囲**：開始日と終了日を選択
   - **モデル**：モデル名のキーワードを入力
   - **トークン名**：トークン名を選択または入力
3. 「クエリ」をクリックすると、一覧がフィルタリング結果に更新されます

![フィルタリング後のログ一覧](/assets/guide/feature-guide/log-filtered.png)

## データ統計

1. 左側ナビゲーションの「データダッシュボード」をクリックするか、`/console` に直接アクセスします

![個人データ統計グラフ](/assets/guide/feature-guide/dashboard-chart.png)

2. データダッシュボードページには日別の API 呼び出し量とクォータ消費トレンドが折れ線グラフまたは棒グラフで表示されます
3. グラフにマウスを重ねると特定の日付の詳細データを確認できます
