トークンは API 認証情報です。各トークンに独自の権限範囲とクォータ制限を設定できます。左側ナビゲーションの「トークン」をクリックするか、`/console/token` に直接アクセスしてください。

![トークン一覧ページ](/assets/guide/feature-guide/token-list.png)

トークン一覧には、作成済みのすべてのトークンの名前、ステータス、使用済みクォータ、残りクォータ、有効期限が表示されます。

## トークンの作成

1. トークン一覧ページの右上にある「トークンを作成」ボタンをクリックすると、作成ダイアログが表示されます
2. トークン名を入力します（用途別に命名することを推奨。例：「本番環境」「テスト用」）
3. 必要に応じて以下のオプションを設定します：

![トークン作成ダイアログ](/assets/guide/feature-guide/token-create.png)

| 設定項目 | 説明 |
| --- | --- |
| 有効期限 | 有効期限を設定。空白または -1 で無期限 |
| 残りクォータ | このトークンが消費できる最大クォータを制限。超過すると自動無効化 |
| 無制限クォータ | 有効にするとクォータ制限なし（アカウント総クォータの制約は受ける） |
| モデル制限 | このトークンで呼び出せるモデルを制限。空白で制限なし |
| IP 許可リスト | このトークンの使用を許可する送信元 IP を制限。空白で制限なし |
| グループ | このトークンが使用するチャンネルグループを指定 |

4. 「送信」をクリックすると、ダイアログに完全なトークンキーが表示されます。**すぐにコピーして保存してください**。ダイアログを閉じると完全なキーは二度と表示されません。

![トークン作成成功、完全なキーを表示](/assets/guide/feature-guide/token-created-key.png)

<Callout type="warn">
  トークンキーは作成時に一度だけ完全表示されます。すぐにコピーして保存してください。トークンキーには完全な API 呼び出し権限があります。他人に漏らしたり、コードリポジトリにコミットしたりしないでください。
</Callout>
