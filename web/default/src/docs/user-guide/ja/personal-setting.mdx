アカウント情報、セキュリティ設定、サードパーティアカウント連携を管理します。ログイン後、右上のアバターをクリックして「個人設定」を選択するか、`/console/personal` に直接アクセスしてください。

## 基本情報

![個人情報ページ](/assets/guide/feature-guide/personal.png)

基本情報セクションでは以下の操作が可能です：

- **ユーザー名の変更**：新しいユーザー名を入力して「保存」をクリック
- **メールアドレスの連携**：メールアドレスを入力し「認証コードを送信」をクリック、認証コードを入力して「連携」をクリック
- **パスワードの変更**：現在のパスワード、新しいパスワード、確認用パスワードを順に入力し「パスワードを変更」をクリック

## 二要素認証（2FA）

2FA を有効にすると、ログイン時にパスワードに加えて認証アプリの動態コードが必要になり、不正アクセスを効果的に防止できます。

![2FA 設定エリア](/assets/guide/feature-guide/2fa-section.png)

設定手順：

1. スマートフォンに認証アプリをインストールします（Google Authenticator または Microsoft Authenticator を推奨）
2. 個人設定ページで「二要素認証」エリアを見つけ、「2FA を有効にする」をクリックします
3. 認証アプリでページに表示された QR コードをスキャンすると、アプリに 6 桁のコードが表示されます

![2FA QR コードダイアログ](/assets/guide/feature-guide/2fa-qrcode.png)

4. アプリに表示された 6 桁のコードを確認フィールドに入力し、「有効化を確認」をクリックします
5. システムがバックアップコードを表示します。スクリーンショットを撮るか書き留めて、すぐに保存してください

<Callout type="warn">
  バックアップコードはダイアログを閉じると二度と表示されません。バックアップコードを紛失し認証アプリも使えない場合は、管理者に 2FA のリセットを依頼する必要があります。
</Callout>

## Passkey パスワードレスログイン

Passkey はデバイスの指紋認証、顔認証、またはハードウェアセキュリティキーでのログインに対応しており、パスワード不要です。

![Passkey 設定エリア](/assets/guide/feature-guide/passkey-section.png)

登録手順：

1. 個人設定ページをスクロールして「Passkey」エリアを見つけます
2. 「Passkey を登録」をクリックします
3. ブラウザがシステム認証プロンプトを表示します。指示に従って指紋、顔認証、またはセキュリティキーで認証を完了します
4. 認証が完了すると Passkey の登録が完了し、次回のログインから使用できます

## サードパーティアカウント連携

GitHub、Discord などのサードパーティアカウントを現在のアカウントに連携すると、パスワードなしでそのアカウントで直接ログインできるようになります。

![サードパーティアカウント連携エリア](/assets/guide/feature-guide/oauth-bind-section.png)

連携手順：

1. 個人設定ページをスクロールして「サードパーティアカウント連携」エリアを見つけます
2. 連携したいプラットフォームの「連携」ボタンをクリックします
3. そのプラットフォームの認証ページにリダイレクトされます。ログインして「認証」をクリックします
4. 認証完了後、自動的に戻り、連携状態が「連携済み」に変わります
