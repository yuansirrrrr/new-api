定価ページにはすべての利用可能なモデルの課金倍率が表示されます。ログイン不要で確認できます。左側ナビゲーションの「定価」をクリックするか、`/pricing` に直接アクセスしてください。

![定価ページ全体](/assets/guide/feature-guide/pricing.png)

ページにはすべての利用可能なモデルが一覧表示されます。各行にはモデル名、入力価格、出力価格が表示されます。

## 定価の確認方法

1. 左側ナビゲーションバーの「定価」をクリック
2. ページにすべての利用可能なモデルの入力価格と出力価格が一覧表示されます
3. 価格はプラットフォームのクォータ単位で表示されます。換算比率は管理者が設定します

## 価格の説明

- **入力価格**：1K 入力トークンあたりのクォータ消費量
- **出力価格**：1K 出力トークンあたりのクォータ消費量
- 実際の消費量 = トークン数 × 対応する倍率
- グループによって異なる課金倍率が適用される場合があります
