New API は、テストや日常使用のために、さまざまな AI チャットアプリケーションへの迅速な設定インポートをサポートしています。

## ワンクリック設定インポート

コンソール→トークンページから、サポートされているチャットアプリケーションにワンクリックで設定をインポートできます。

![ワンクリック設定インポート](/assets/guide/import-chat-config.png)

### 使用方法

1. トークンリストで使用したいトークンを見つける
2. トークンの右側にある「インポート」または「設定」ボタンをクリック
3. ターゲットチャットアプリケーションを選択
4. システムが自動的に対応するアプリケーションにリダイレクトし、設定情報を入力

## Lobe Chat

<Callout type="info">
  Lobe Chat は、複数の大規模言語モデル（LLM）とプラグイン拡張をサポートするオープンソースのマルチモーダル会話アプリケーションです。テキスト会話だけでなく、画像や音声などのマルチモーダルインタラクションもサポートし、個人アシスタント、知識Q&A、コンテンツ作成など、さまざまなシナリオに適しています。Lobe Chat はシンプルで直感的なインターフェースを備え、マルチデバイス同期をサポートし、ユーザーはニーズに応じてモデルとプラグインをカスタマイズして、会話機能を柔軟に拡張できます。開発者でも一般ユーザーでも、簡単に始めて、インテリジェントな会話がもたらす効率的な体験を楽しむことができます。
</Callout>

<Callout type="warn">
  Lobe Chat は現在 New API からのワンクリックインポートをサポートしておらず、手動設定が必要です
</Callout>

### 設定手順

1. Lobe Chat アプリケーションを開く
2. 設定ページに移動し、「言語モデル」設定エリアを見つける
3. 以下の情報に従って設定を入力：

![Lobe Chat 設定入力 - ステップ1](/assets/guide/lobechat-1.png)

- **API アドレス**：New API プラットフォームの Base URL を入力
- **API Key**：New API で作成したトークンを入力

![Lobe Chat 設定入力 - ステップ2](/assets/guide/lobechat-2.png)

4. 「保存」をクリックして設定を完了
5. モデル選択で New API が提供するすべての利用可能なモデルが表示されます

## AI as Workspace

<Callout type="info">
  AI as Workspace は、人工知能機能をワークスペースに統合する革新的な方法です。AI アシスタントを日常のオフィスワーク、コラボレーション、知識管理などのシナリオと深く組み合わせることで、ユーザーはより効率的に情報を処理し、反復タスクを自動化し、インテリジェントな提案を受けて、全体的な作業効率と体験を向上させることができます。
</Callout>

<Callout type="info">
  New API からのワンクリックインポートをサポート
</Callout>

### 使用方法

1. New API トークンページで「AI as Workspace にインポート」をクリック
2. システムが自動的に AI as Workspace にリダイレクトし、設定を入力
3. 設定情報を確認して使用開始

## AMA 問天

<Callout type="info">
  AMA 問天は New API が提供するインテリジェント Q&A アシスタントで、マルチターン会話と複雑な質問への回答をサポートしています。ユーザーは自然言語で AMA 問天と交流し、知識、技術サポート、またはビジネスコンサルティングを取得して、コミュニケーション効率と体験を向上させることができます。
</Callout>

<Callout type="warn">
  AMA 問天アプリを先にインストールする必要があります
</Callout>

### 使用方法

1. アプリストアから AMA 問天アプリをダウンロードしてインストール
2. New API トークンページで「AMA 問天にインポート」をクリック
3. アプリが自動的に開いて設定を完了
4. AI アシスタントとの会話を開始

## OpenCat

<Callout type="info">
  OpenCat は、複数の大規模言語モデル統合をサポートするオープンソースのマルチプラットフォーム AI チャットクライアントです。ユーザーはシンプルなインターフェースを通じて AI と自然言語で会話でき、日常のコミュニケーション、知識Q&A、コンテンツ作成などのさまざまなシナリオに適しています。OpenCat はマルチデバイス同期をサポートし、設定が柔軟で、個人やチームでの使用に適しています。
</Callout>

<Callout type="warn">
  OpenCat アプリを先にインストールする必要があります
</Callout>

### 使用方法

1. アプリストアから OpenCat アプリをダウンロードしてインストール
2. New API トークンページで「OpenCat にインポート」をクリック
3. アプリが自動的に開いて設定を完了
4. OpenCat でモデルを選択して会話を開始

## 他のアプリケーションの手動設定

ワンクリックインポートをサポートしていないアプリケーションの場合、手動で設定できます：

### 設定情報

New API から以下の情報を取得：

| 設定項目 | 説明 | 取得場所 |
| --- | --- | --- |
| API Base URL | API インターフェースアドレス | プラットフォームホームページまたはトークンページ |
| API Key | トークン秘密鍵 | トークン作成時に表示（一度のみ） |
| 利用可能なモデル | サポートされているモデルリスト | 個人設定→利用可能なモデル または 価格ページ |

### 一般的な設定手順

1. ターゲットアプリケーションで「設定」または「構成」エントリを見つける
2. 「カスタム API」または「OpenAI 互換インターフェース」設定エリアを見つける
3. API Base URL と API Key を入力
4. 設定を保存して使用開始

<Callout type="info">
  OpenAI API をサポートするほとんどのアプリケーションは、カスタム API アドレスを入力することで New API と統合できます
</Callout>
